# Python

## O que faz: 

Solicita a idade do usuário e classifica se ele é maior ou menor de idade.

## Destaque Técnico: 

Implementação de um loop while True para garantir que o programa não feche se o usuário digitar algo inválido (como uma letra).

## Conceitos Aplicados: 

Tratamento de exceções (ValueError), Condicionais (if/elif/else) e Loops de repetição.
